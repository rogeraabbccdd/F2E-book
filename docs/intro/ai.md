## MCP
MCP (Model Context Protocol) 是一個開放標準，讓 AI 模型能夠存取外部資料與工具

:::tip TIP
每個 MCP 的系統需求不同，常見的需求有
- [Node.js](https://nodejs.org/)
- [Python](https://www.python.org/)
- [Docker](https://www.docker.com/)
:::

蒐集整理
- [Smithery AI](https://smithery.ai/)
- [Awesome MCP Servers](https://github.com/punkpeye/awesome-mcp-servers/blob/main/README.md)

## Skills
Skills 定義 AI 的工作行為說明，像是 SOP

蒐集整理
- [skills.sh](https://skills.sh/)

推薦
- [find-skills](https://skills.sh/vercel-labs/skills/find-skills) 讓 AI 自己搜尋技能
- [web-design-guidelines](https://skills.sh/vercel-labs/agent-skills/web-design-guidelines) 網頁設計指南
- [brainstorming](https://skills.sh/obra/superpowers/brainstorming) 讓 AI 腦力激盪後才產生內容
- [seo-audit](https://skills.sh/coreyhaines31/marketingskills/seo-audit) 檢查並提出 SEO 改善建議
- [find-docs](https://skills.sh/upstash/context7/find-docs) 搜尋 [Context7](https://context7.com/) 上的最新文件
- [excalidraw-diagram-generator](https://skills.sh/github/awesome-copilot/excalidraw-diagram-generator) 繪製 [Excalidraw](https://excalidraw.com/) 圖表
- [frontend-design](https://skills.sh/anthropics/skills/frontend-design) 擺脫 AI 味，產生有設計感的版面
